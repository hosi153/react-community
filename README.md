# React Community

React와 Tailwind CSS를 사용한 커뮤니티 프로젝트입니다.

## 기술 스택

- **React** 19.1.0
- **Vite** 6.3.5
- **Tailwind CSS** 4.1.10
- **TypeScript**
- **ESLint**

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 프리뷰

```bash
npm run preview
```

### 린트

```bash
npm run lint
```

## 프로젝트 구조

```
src/
├── App.jsx          # 메인 앱 컴포넌트
├── main.jsx         # 앱 진입점
└── index.css        # Tailwind CSS 설정
```
